# Financial-Performance-Dashboard
Interactive Financial Performance Dashboard built in Excel using Pivot Tables, Slicers, and Charts to analyze Revenue, Expense, and P&amp;L.

An interactive Excel dashboard that provides a complete 360° view of business financial performance — analyzing Revenue, Expense, Profit & Loss, and Department-wise trends with dynamic filtering.

##Dataset Used
 - <a href="https://github.com/jaykavitake/Financial-Performance-Dashboard/blob/main/Finance_dashboard.xlsx">DataSet</a>


##Questions (KPIs)
The following business questions were analyzed and answered through this dashboard:

📊 Compare the Revenue and Expense using a single chart
📅 Which month got the highest Revenue and Expense?
⚖️ What is the Profit & Loss trend across all 12 months?
🏢 Which Department is contributing maximum to the Revenue?
💳 Which Payment Method is used the most?
📆 Which Weekday generates the highest Revenue?
📂 What are the different Revenue categories and their contribution?
📉 What is the highest and lowest Revenue recorded?
🏆 Which Department has the best Revenue vs Expense ratio?
📦 Percentage of Total Revenue contributed by each Department
